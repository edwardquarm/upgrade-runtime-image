# Upgrade Runtime Image Helper

This repo contains two helper scripts:

- `upgrade-runtime/upgrade-runtime.sh`: Upgrade KServe InferenceServices to the latest ServingRuntime from templates.
- `deprecated-and-removed-runtime/serving-runtime.sh`: Scan runtimes, show templates and images, and flag removed/deprecated templates.
